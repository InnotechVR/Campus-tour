Recruitment / VR booth

Campus tour · Street View

Paste a Google Maps API key with the Maps JavaScript API enabled, then start. Look around by dragging; on a phone-in-headset, turning the device moves the view. Tap a stop below to jump between locations.

Note: Google’s Street View widget renders as a mono panorama — drag/phone-tilt to look. It is not stereo head-tracked VR (Google doesn’t expose that to the browser). For true head-tracked stereo VR use self-hosted 360° photos instead.

Get a key at console.cloud.google.com/google/maps-apis, enable “Maps JavaScript API,” and restrict it to your domain before public use.
